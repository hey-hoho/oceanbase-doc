# USEC_TO_TIME

## 声明

```javascript
USEC_TO_TIME(usec)
```

## 说明

将 `usec` 值转换为 `TIMESTAMP` 类型值。

`usec` 表示距离 `1970-01-01 00:00:00.000000` 的微秒数，并考虑时区信息。

## 示例

```javascript
obclient> SELECT SEC_TO_TIME('54925');
+----------------------+
| SEC_TO_TIME('54925') |
+----------------------+
| 15:15:25.000000      |
+----------------------+
1 row in set
```
